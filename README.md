## ✨ Key Features
학교 근처 찐맛집 데이터 베이스를 기반으로 실패 없는 메뉴 선정을 도와드립니다.

### 1. 에리카 맞춤형 카테고리
- **7가지 핵심 테마:** 면, 한식, 중식, 일식, 양식, 패스트푸드, 카페/디저트 중 그날의 기분에 맞는 카테고리를 선택하세요.
- 학교 근처 식당들의 위치를 고려하여 선정하였습니다.

### 2. 운명의 룰렛 (Roulette)
- 선택한 카테고리 내에서 무작위로 메뉴를 선정합니다.
- 친구들과 "이거 나오는 거 먹으러 가자!" 내기를 걸고 룰렛을 돌려보세요.

### 3. 학교 앞 맛집 네비게이터 (Store List)
- 메뉴만 알려주고 끝내지 않습니다. **학교 근처(정문, 쪽문 등)**에서 해당 메뉴를 파는 식당 리스트를 보여줍니다.
- **네이버 지도 연동:** 버튼 한 번이면 식당 위치와 리뷰를 바로 확인할 수 있어 공강 시간을 효율적으로 쓸 수 있습니다.

## 💬 Our Story
**"정문까지 걸어 나갔는데, 결국 편의점 김밥 드셔보신 적 있나요?"**

저희 개발팀 3명은 매일 점심시간마다 셔틀콕 앞이나 정문에서 "뭐 먹을래?" "아무거나..."라는 대화를 무한 반복하던 평범한 에리카 학생들이었습니다. 학교 앞 맛집은 많은데, 막상 선택의 순간이 오면 자꾸 선택을 떠넘기게 되었습니다.

**"우리 학교 학생들의 고민은 우리가 해결하자!"**

이 프로젝트는 에리카 학우들의 사소하지만 확실한 스트레스를 해결하기 위해 시작되었습니다. "오픈소스개발과 활용"이라 과목에서 배운 사소한 지식을 바탕으로, 우리 학교 사람들에게 실질적인 도움이 되는 서비스를 만들고 싶었습니다.

저희가 엄선한 학교 앞 맛집 데이터와 룰렛 기능을 통해, 에리카 학우분들의 점심시간이 고민 대신 **설렘과 맛있는 즐거움**으로 채워지길 바랍니다.

오늘도 맛점하세요! 🦁💙
